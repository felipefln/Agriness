CMS-Agriness
A Simple Content Management System (CMS)

Technologies used:
Django (Web framework)
Django Rest Framework - REST API
Postgres to database 
Swagger for creating documentation

File structures:
core - System app folder
core.migrations - Migration files for database creation
core.models - Definition of tables
core.serializer - Class for data serialization
core.viewset - Viewset class for displaying data (API)

settings.py - Django configuration file
urls.py - Django and API route file
Procfile - Heroku deploy config file
start.bat - Environment variables (Windows format)/
